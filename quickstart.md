# Quickstar

Criar ambiente virtual

```python
python3 -m venv venv
```

Ativar 
```python
source venv/bin/activate
```
instalar dependencias
```pip
pip3 install -r requirements.txt
```
iniciar server

```pip
python3 manage.py runserver
```

tests
```python
python3 manage.py test
```
